1) RUNNING EXAMPLES  k_means
    a) change directory to source and unzip octavefiles.zip
    b) run octave(or matlab-not tested) and call function main!
        Note: you have to start octave(or matlab) in the directory where you unzipped the octavefiles.zip
    c) if you specify argument other than 0 than it will start plotting.
       You should hit enter in the octave window to move to next step.
